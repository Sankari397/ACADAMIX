
🎓 ACADEMIX – Mentor Mentee Tracking System
📌 1. Project Title

Academix: Real-Time Mentor–Mentee Student Performance Tracking System

📖 2. Introduction

Academix is a web-based application designed to improve communication and academic monitoring between mentors and mentees in a college environment. Traditional systems rely on manual tracking, which is time-consuming and error-prone. This system digitizes the process and provides real-time updates of student performance.

The application ensures transparency, efficiency, and accuracy in tracking academic records such as internal marks, assignments, and attendance.

🎯 3. Project Objective

The main objectives of Academix are:

To develop a real-time student performance tracking system

To provide role-based access for Admin, Mentor, and Mentee

To automate academic evaluation

To improve mentor–mentee communication

To reduce manual errors in academic tracking
